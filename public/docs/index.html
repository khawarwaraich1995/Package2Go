Introduction

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

http://pkg2day.com

Authenticating requests

This API is not authenticated.

Client Addresses

API endpoints for managing clients addresses

Add new address Endpoint

This endpoint allows you to add commonly used addresses for user. To add new addresses you need access_token generated by this authentication endpoints.

Example request:
curl --request POST \
    "http://pkg2day.com/api/v1/client/store-address" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"first_name\": \"John\",
    \"last_name\": \"Panda\",
    \"company_name\": \"Google LLC\",
    \"street_address\": \"Jinnah Avenue near Zero Point\",
    \"coordinates\": \"{\\\"lat\\\": \\\"57886\\\", \\\"lng\\\": \\\"23112\\\"}\",
    \"zip_code\": 64200,
    \"city\": \"Islamabad\",
    \"street_address2\": \"Front of Jinnah Park\"
}"

const url = new URL(
    "http://pkg2day.com/api/v1/client/store-address"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "first_name": "John",
    "last_name": "Panda",
    "company_name": "Google LLC",
    "street_address": "Jinnah Avenue near Zero Point",
    "coordinates": "{\"lat\": \"57886\", \"lng\": \"23112\"}",
    "zip_code": 64200,
    "city": "Islamabad",
    "street_address2": "Front of Jinnah Park"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$response = $client->post(
    'http://pkg2day.com/api/v1/client/store-address',
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'json' => [
            'first_name' => 'John',
            'last_name' => 'Panda',
            'company_name' => 'Google LLC',
            'street_address' => 'Jinnah Avenue near Zero Point',
            'coordinates' => '{"lat": "57886", "lng": "23112"}',
            'zip_code' => 64200,
            'city' => 'Islamabad',
            'street_address2' => 'Front of Jinnah Park',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));

Example response (200):


{
     "status": true,
     "message": "Address is created successfully!",
 }

Example response (401):


{
    "message": "Unauthenticated."
}
